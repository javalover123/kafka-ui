## Table of contents

## Getting started

Install packages

```
npm install
```

Set correct URL to your API server in `.env`.

```
REACT_APP_API_URL=http://api.your-kafka-rest-api.com:3004
```

Start JSON Server if you prefer to use default full fake REST API.

```
npm run mock
```

Start application

```
npm start
```
